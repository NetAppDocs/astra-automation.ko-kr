===== Astra 계정 자격 증명을 가지고 있어야 합니다

Astra 웹 사용자 인터페이스에 로그인하고 API 토큰을 생성하려면 Astra 자격 증명이 필요합니다. Astra Control Center를 사용하면 이러한 자격 증명을 로컬로 관리할 수 있습니다. Astra Control Service를 사용하면 계정 자격 증명이 * Auth0 * 서비스에 의해 유지됩니다.
